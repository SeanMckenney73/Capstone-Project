# League of Data

A wikipedia style application for information regarding league of legends

## Description

In this application you can view information about the 3 main components of league of legends that player should know in-game:
Champion information: Ability information and Cooldowns
Rune information: What they do, if they have cooldowns, and how they affect abilities and items
Item information: An item's passive and/or active ability, what stats they give, and how much they cost

## Getting Started

### Dependencies

Dependencies needed (all via NPM):
for backend folder
    - @supabase/supabase-js
    - cors
    - .env
    - express.js

for LOD-react folder
    - react.js
    - bootstrap/react-bootstrap
    - react-dom
    - react-router-dom

### Installing

Download the code via the current repository

### Executing program

open 2 terminals
in the first terminal cd into LOD-React and run 'npm run dev'
in the second terminal cd into backend and run 'npm start'

```
npm run dev

npm start

```
